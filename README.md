# Project- Recipe Book and Shopping List Application

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.0.8.

# Project Description
The project is designed to view a list of recipes and their description, manage the shopping list for the recipes. Users can buy the ingredients for the recipes through the application.
